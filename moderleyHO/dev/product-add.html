How to add products into game

warning

Having issues? Please STOP and message a developer to avoid causing damage to the game.

note

The file needed to use this in a game is now available in the development team files tab.

Step 1 - Checking the product​

You will require 1 tool and one model. The tool should be similar to below.

The model can just be a model, nothing fancy here. It doesn't even need to be grouped or anything. You can have as many or as little as you want. It can be as simple as the below

Once completing that you can move onto the next step.

Step 2 - Setting up the proximity prompt​

In the explorer, expand the workspace tab and find the folder called "ProductPrompters"

Inside of the product prompter is a click detector with an attribute.

The attribute 'ProductName' should be set to something unique, yet easily memorable as it will be required later.

Step 3 - Configuring the tool​

In ServerStorage a folder named Products should be located. The tool that was made in step 1 should be pasted into this folder.

You must make sure that the name of this tool matches the attribute set in the clickdetector in step 2. If not it will not work.

Conclusion​

If all the above steps have been followed. The product should now be clickable.
